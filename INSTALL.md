# Instruções de Instalação

## 📱 iPhone (iOS) - Safari

O Safari no iOS não suporta instalação automática de PWAs. Siga estes passos:

1. **Abra o site no Safari**
2. **Toque no botão "Compartilhar"** (ícone de quadrado com seta para cima)
3. **Role para baixo e selecione "Adicionar à Tela de Início"**
4. **Toque em "Adicionar"** no canto superior direito
5. O app aparecerá na sua tela inicial!

### Dicas para iOS:
- Funciona no iOS 11.3 ou superior
- O app funciona offline após a primeira visita
- Para atualizar, arraste para baixo na tela do app

---

## 🤖 Android - Chrome, Edge, Samsung Internet, Firefox

No Android, a instalação é mais simples:

### Chrome/Edge:
1. Abra o site no navegador
2. Toque no menu (3 pontos) ou aguarde o banner de instalação
3. Selecione "Instalar aplicativo" ou "Adicionar à tela inicial"
4. Confirme a instalação

### Samsung Internet:
1. Abra o site
2. Toque no menu (3 linhas)
3. Selecione "Adicionar página à" → "Tela inicial"

### Firefox:
1. Abra o site
2. Toque no menu (3 pontos)
3. Selecione "Instalar"

---

## 💻 Desktop (Chrome, Edge)

1. Abra o site no Chrome ou Edge
2. No canto direito da barra de endereço, clique no ícone de instalação
3. Ou use o menu (⋮) → "Instalar Quem Visita a Terra"

---

## ⚠️ Notas Importantes

- **iPhone**: A Apple não permite instalação automática. Use sempre o método manual pelo Safari.
- **Android**: Funciona em qualquer navegador moderno.
- **Offline**: Após a instalação, o app funciona sem internet.
- **Atualizações**: O app atualiza automaticamente quando você o abre com internet.
